# Sales Intelligence Dashboard with Generative AI

## Project Overview

An end-to-end Sales Intelligence Dashboard built using SQL, Python, Machine Learning, Streamlit, and Generative AI.

The project helps businesses monitor sales performance, profit trends, target achievement, customer insights, and AI-generated business recommendations through an interactive dashboard.

---

## Business Problem

Business teams often spend significant time preparing sales reports, identifying profit issues, and finding growth opportunities manually.

This project automates sales analysis and provides AI-powered insights to support faster business decisions.
