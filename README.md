# Sales Intelligence Dashboard

## Project Overview
An end-to-end Sales Intelligence Dashboard built using SQL, Python, Pandas, Machine Learning and Streamlit.

## Features
- Sales Analysis
- Profit Analysis
- Target vs Actual Analysis
- Sales Forecasting
- Anomaly Detection
- AI Business Questions
- Interactive Dashboard

## Technologies Used
- SQL
- Python
- Pandas
- Scikit-Learn
- Streamlit
- GitHub

## Project Files
- app.py
- sales_intelligence.sql
- List_of_Orders.csv
- Order Details.csv
- Sales target.csv
